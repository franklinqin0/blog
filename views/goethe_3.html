# Können, Kennen, Wissen (3 Know's)

Können is a modal verb (know how to do sth). e.g.

  • Ich kann in Ruhe lesen.
  • Ich kann Deutsch.

wissen is used if seeing clauses (dass, wie, wann, wo, wer, warum); kennen is used if seeing nouns.
