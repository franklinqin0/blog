x to the n

franklinqin0 2020-02-17 leetcode

Implement pow(x, n), which calculates x raised to the power n ().

Note:

  • -100.0 < x < 100.0
  • n is a 32-bit signed integer, within the range [, − 1]

Or could store the common half:

def myPow(self, a, b):
    if b == 0: return 1
    if b < 0: return 1.0 / self.myPow(a, -b)
    half = self.myPow(a, b // 2)
    if b % 2 == 0:
        return half * half
    else:
        return half * half * a
