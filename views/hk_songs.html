        香港歌曲

        franklinqin0 2020-05-15 life

        一些十分喜爱的粤语歌曲。

        # 许冠杰

        # Beyond

        Beyond是殿堂级的HK乐队，他们的歌我喜欢的实在真的太多了。他们的音乐代表热血、代表青春。高中时我听了他们全部的歌，后来在大学听周杰伦比较多，Beyond就没怎么听了。但一次与朋友们卡拉OK时，尽管多年没唱，放完前奏依然想不起调，但歌词一出来便脱口而出。这就是埋藏心底的旋律吧！

        Note: 有名的歌真的好多是标题四个字的。

        # 再见理想

        五年总结，继续革命

        真的爱你*：送给母亲的情书 光辉岁月*：送给Nelson Mandela、自由、平等、博爱、光辉人性的情书 喜欢你*：送给过去恋人的情书 情人*：即将去日本送给粉丝的情书 海阔天空*：十年总结，终有一别

        家驹有情，但不痴情。他追求理想，但知道什么时候与商业妥协。让我们欣慰的是，他并不孤单，因为“摇滚精神，永垂不朽”。我相信他的歌仍会继续被传唱，十年、二十年、三十年...

        # 张学友

        # 陈奕迅

        # 杨千嬅

        十分喜欢杨千嬅唱歌的力道！如四两拨千斤，又如轻风拂面，听后仍余音绕梁、回味悠长。

        # 小城大事

        林夕写的好一首失恋催泪苦情歌！

        吻下来 豁出去 这吻别似覆水
        再来也许要天上团聚

        我下来 你出去 讲再会也心虚
        我还记得到天上团聚

        吻下来 豁出去 从前多么登对
        你何以双眼好像流泪

        # 可惜我是水瓶座

        这首歌每句都不难理解，但连在一起就有点难解了。这种歌者的内心独白，我不明白，毕竟我不是水瓶座。

        如何笨到底 但到底还是我
        谁人待我好 待我差 太清楚
        想继续装傻 却又无力受折磨
        心里羡慕有些人 盲目到不计后果

        # 友谊万岁

        虽然这首骊歌十分有名，但新改的歌词与传统版的相比，寄托更多毕业离别的愁绪和对未来的祝福与展望，更贴近电影。

        诚意送上祝福 盼好风会 常为你吹
        此际伸手笑握 纵声歌唱 在校园里
        不理日后如何 前途赶上 步步进取
        今天双手放开 再握之际 又在何处

        # 陈百强

        Danny的歌声纯净清亮，能带来温暖和希望。歌词很有古典的韵味，有点邓丽君的感觉。可惜英年早逝，坟墓和家驹一样，在将军澳华人永远坟场。

        # 偏偏喜欢你

        光看名字，不知道这首歌原来是失恋后必备的苦情歌。但曲调优美、婉转动人，不禁也因歌者的痴心而落泪。

        爱已是负累
        相爱似受罪
        心底如今满苦泪
        旧日情如醉 此际怕再追
        偏偏痴心相见你

        为何我心分秒想着过去
        为何你一点都不记起
        情义都失去恩爱都失去
        我却为何偏偏喜欢你

        # 一生何求

        一生苦短，一生何求？我们都是迷惘的追梦人，猛然回头，却发现自己在追逐名利中迷失，失去了最珍惜的拥有。看了许多名人的传记，但如何才能过好这一生？如果追求会让你失去所有，你仍会追求吗？

        一生何求 常判决放弃与拥有
        耗尽我这一生 触不到已跑开

        一生何求 迷惘里永远看不透
        没料到我所失的 竟已是我的所有

        # 喝采

        在你的生命中，无论挫折与成功，我都“愿将一腔热诚给你，常为你鼓舞“。

        似朝阳正初升
        你要自信有光明前路
        愿知生命诚可贵 能为你鼓舞

        路上我愿给你轻轻扶
        你会使我感到好骄傲
        幸福欢乐陪着你去找一生美好

        # 周星驰

        等等，星爷什么时候是歌星了？他虽没出过专辑，但电影里许多音乐的场景都令人拍案叫绝。这是他的歌唱合集。下面选两首我个人喜欢的。

        # 李香兰

        同一首《李香兰》，张歌神唱得哀婉，凌凌漆唱得喜感。在星爷的多次歌唱中，我最喜欢的便是这首。

        # California Dreamin'

        原版插曲在王家卫导演的《重庆森林》中多次出现，表达拥闹的城市中个人的迷离和人与人之间的隔阂。

        在星爷的《少林足球》中，用了这首歌的伴奏，以戏谑的唱腔为少林功夫站台。虽然十分喜感，可惜酒吧的观众不买账，把歌者用啤酒瓶打了一顿。

        # 儿歌

        有几首蛮喜欢的粤语儿歌，他们带给我温暖和力量。

        # 月光光

        家驹在世时，Beyond在红馆第一场演唱会是1991年的生命接触。在众多曲目中，只有这一首是非原创的。

        《月光光》大概是粤语儿歌中最著名的一首了，许多广东人即使不记得歌词也能哼出曲调。歌词版本非常多，但都以“月光光 照地堂”开头。有一次与YG在花城汇听到一个粤剧演员唱的就比Beyond的版本长许多。

        # 小太阳

        把太阳比作红花和南瓜，真是拥有无尽的童趣！太阳也会倦，但明天依旧来。

        太阳倦了便回家
        夜里休息少惊怕
        明晨月儿落旭日重来
        依旧往上爬

        # 世界真是小

        这首歌有多种语言的版本。很遗憾，听国语版就是没有粤语版那样震撼，那是种听到贝多芬第九交响曲中席勒的《欢乐颂》大合唱时的震撼。人类是命运共同体，太多时候我们都自私自利、损人利己，但为什么不“相助共济”、合作共赢，让爱与和平便会充满大地呢？

        人类常欢笑不要眼泪掉
        时时怀希望不必心里跳
        在那人世间相助共济
        应知人间小得俏

        世界真是小小小
        小得真奇妙妙妙
        实在真是小世界
        娇小而妙俏
