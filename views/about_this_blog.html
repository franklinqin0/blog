            # Why Not Social Networks

            In recent years, people don't bother w/ creating their personal websites but spend more time on Instagrm/WeChat/Twitter/Weibo.

            The Internet is becoming more stupid and arousing. Sexy pictures and funny videos sell.

            Why not just post WeChat moments w/ no more than 9 pictures and interact w/ friends thru a like click? I once seriously doubt the value of creating this site. But I made up my mind to continue this site after reading this popular essay, and I quote:

            # Built in Jekyll

            In early 2018, I used Jekyll as the static site generator to construct a MVP.Gradually, I found it hard to extend, such as multi-language support, table of contents on sidebar and dark mode. It's not impossible to add an essential feature, but my hands would be quite dirty (see Docker doc). I had a list of stuff I want my blog to have but hard to achieve in reasonable amount of time if keep using Jekyll.

            # Transformed to VuePress

            In early 2020, while searching for alternatives, I was quite excited to discover *VuePress*, which is based on Vue.js, a framework I've always been quite curious about, after having learned some Angular.js and React.js. Using VuePress, bloggers can build a SPA (blazing fast when routing w/i the site) w/ dynamic Vue components, rather than a static site compiled by Jekyll.

            My passion faded a bit when I found that VuePress is not as supportive for blog as for documentation, but I searched "VuePress blog theme" on GitHub and found the blog theme reco. I marvelled at its simplicity and elegance. It has most features I need, if not more! So I spent a few days during 2020 Spring Festival to make some changes most in structure and few in blog posts to transport my old blogs from Jekyll to vuepress-theme-reco.
